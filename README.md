# FIFA World Cup 2026 — ML Prediction Model

A machine learning system that predicts the winner of the 2026 FIFA World Cup using
historical match data, FIFA rankings, and EA Sports FC 24 player ratings.
Predictions are generated by running 10,000 Monte Carlo simulations of the full
48-team tournament bracket.

## How it works

### Data pipeline
- **49,477 international match results** from 1872 to June 2026 (Kaggle)
- **FIFA World Rankings** snapshots from 1992 to June 2024
- **EA Sports FC 24 player ratings** — 18,350 players across 160 nations

### Feature engineering
For each match, the model uses:

| Feature | Description |
|---------|-------------|
| `rank_diff` | FIFA ranking difference between teams |
| `points_diff` | FIFA points difference |
| `squad_avg_diff` | Difference in average squad rating (top 23 players) |
| `top11_diff` | Difference in starting XI quality |
| `star_player_diff` | Difference in best player rating |
| `depth_diff` | Bench quality difference |
| `form_diff` | Difference in recent win rate (last 20 matches) |
| `is_neutral` | Whether the match is on neutral ground (always 1 for WC) |

### Model selection
Three classifiers were trained and evaluated on a time-ordered train/test split:

| Model | Accuracy |
|-------|----------|
| **Logistic Regression** | **60.4%** |
| XGBoost | 58.5% |
| Random Forest | 55.8% |

Logistic Regression outperformed tree-based models on this dataset because
squad quality and ranking signals have a predominantly linear relationship
with match outcomes. Football prediction baseline (random) = 33.3%.

### Tournament simulation
The trained model generates win/draw/loss probabilities for any matchup.
These probabilities feed a Monte Carlo simulator that:
1. Simulates all 72 group stage matches with probabilistic outcomes
2. Determines group winners, runners-up, and 8 best third-place teams
3. Simulates the Round of 32 through the Final
4. Repeats 10,000 times and aggregates results into win probabilities
